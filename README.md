## Overview

This project is a testament to my passion for API and web development. Using Next.js, TypeScript, and Tailwind CSS, I've created a captivating space to display various car models with advanced filtering and seamless pagination.

## Features

- **Visually Appealing Home Page:** Showcases a diverse range of cars fetched from a third-party API, providing a captivating introduction to the world of automobiles.

- **Exploration and Filtering:** Explore a wide variety of cars from around the world, utilizing a search and filter system based on criteria such as model, manufacturer, year, fuel type, and make.

- **Server-Side Rendering Transition:** Seamlessly transition from client-side rendering to server-side rendering, enhancing performance and providing a smoother browsing experience.

- **Pagination:** Easily navigate through a large dataset of cars, allowing users to explore multiple pages effortlessly.

- **Metadata Optimization and SEO:** Optimize metadata for car listings, enhancing search engine optimization (SEO) and ensuring better visibility on search engine results pages.

- **TypeScript Types:** Utilize TypeScript to provide robust typing for enhanced code quality and better development.

- **Responsive Website Design:** The website is designed to be visually pleasing and responsive, ensuring an optimal user experience across various devices.

## Technologies Used

- **Next.js:** Leveraging server-side rendering capabilities for enhanced performance.

- **TypeScript:** Providing strong typing for improved code quality.

- **Tailwind CSS:** Styling the website with a modern and efficient utility-first CSS framework.
